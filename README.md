# Finance RBAC Backend

A Django-based backend system for managing financial records with role-based access control.

## Features

- User roles (Viewer, Analyst, Admin)
- Financial records CRUD
- Filtering support
- Dashboard analytics
- RBAC enforcement
- PostgreSQL + Docker
